# FAKE-SPAM

## WhatsApp tool (بدون توكن)

تم تعديل الأداة بحيث تشتغل **بدون توكن**.

- فقط أدخل رقم الشخص.
- الأداة تفتح رابط واتساب مع الرسالة الجاهزة.
- الإرسال الفعلي يتم من داخل واتساب.

## تشغيل سريع

```bash
python3 whatsapp_sender.py --to +9647734549509
```

## خيارات إضافية

```bash
python3 whatsapp_sender.py \
  --to +9647734549509 \
  -m "السلام علييكم" \
  --count 3 \
  --delay-seconds 0.5
```

## ملاحظات

- لا يحتاج `token` ولا `phone_number_id`.
- الرقم يجب يكون بصيغة دولية.
